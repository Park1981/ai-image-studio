활성 카드 효과 비교 — A · B · E

"활성 시 인물 이미지가 더 보였으면 좋겠다"는 오빠 의도에 대한 3 옵션 시안. 동일한 생성 패널을 옵션별 CSS 만 다르게 적용 — 카드 토글 켜고 끄며 차이 직접 비교.

  • 옵션 A (높이 증가) — 활성 시 카드 padding 14px → 50px · 인물 풀 노출 ★★★★★ / 단점: 패널 점프 (다른 카드 + CTA 위치 변동)
  • 옵션 B (mask 만 확장) — 활성 시 mask 좌측 35% → 15% · 점프 X / 단점: 인물 절반만 노출 (얼굴 + 어깨)
  • 옵션 E (하이브리드) — 활성 시 padding 14px → 26px + mask 35% → 25% · A 의 50% 임팩트 + 단점 거의 X

비교 방법 — AI 카드 클릭 ↔ Claude 카드 클릭 등으로 토글 켜고 끄면서 옆 옵션과 비교하기. 사이즈 카드는 항상 활성이라 옵션 별로 정적 차이만 노출.

💡 의사 결정 포인트 — 시각 임팩트 vs 안정감 트레이드오프:
A = 임팩트 ★★★★★ / 안정감 ★★☆☆☆
B = 임팩트 ★★★☆☆ / 안정감 ★★★★★
E = 임팩트 ★★★★☆ / 안정감 ★★★★☆ (균형)
