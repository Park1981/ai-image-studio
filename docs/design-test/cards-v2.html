V5 톤 5 패널 풀 시안

모든 패널 V5 Aurora Glass 톤 · 시그니처 컬러 6종 · 카드 전체 클릭 · hover 툴팁 · 카드 일관성.

  • 생성 (Generate) — 프롬프트 → Qwen Image · AI 보정 · Claude 조사 · 퀄리티 모드 · 사이즈
  • 수정 (Edit) — 원본 + 수정 지시 → Qwen Image Edit · AI 보정 (필수) · 자동 평가 · 퀄리티 · 추가 참조 (참조 ON 시 sub-section)
  • 분석 (Vision Analyze) — 단일 이미지 → 영문/한글 설명 (40~120 단어) · 매우 간소 (이미지 + CTA)
  • 비교 (Vision Compare) — 이미지 A·B → 5축 비교 · A↔B 스왑 + 비교 지시 (선택)
  • 영상 (Video Generate) — 원본 + 영상 지시 → 5초 MP4 (LTX-2.3 i2v) · AI 보정 · 해상도 슬라이더 · 퀄리티 · 성인 모드 (NEW crimson 시그니처)

시그니처 컬러 6종 — AI=violet/blue · Claude/자동평가=amber/orange · 퀄리티=lime/cyan · 사이즈/추가참조=rose/pink · 성인=crimson/red.

인터랙션 데모 — 모든 변형:

  • 카드 클릭 → ON/OFF 토글 (V3a/V3b 는 스위치 클릭과 동일 / V3c 는 유일한 클릭 방법)
  • instant/thinking 클릭 → 카드 토글 영향 없이 thumb 슬라이드 (e.stopPropagation())
  • 비율 카드 클릭 → 4 가운데 active 변경
  • 비율잠금 클릭 → 잠금/해제 (e.stopPropagation 으로 사이즈 카드 영향 X)
