CTA 디자인 변형 비교

현재 cards-v2.html 의 단순 파란색 pill CTA 를 6 가지 컨셉으로 다양화. 호버해서 모션 + 디테일 확인.

  • A · Editorial Magazine — 검정 + Fraunces italic + 골드 라인. Vogue Korea 톤. 미니멀 + 럭셔리 정체성.
  • B · Aurora Glass Gradient — violet→blue 시그니처. 시스템 카드 톤 매칭. light sweep 모션.
  • C · Holographic Shimmer — 5색 메탈릭 시머. 마법/AI 정체성 maximalist. 무한 애니메이션.
  • D · Spotlight Pulse — radial indigo + 펄스 + sparkle 회전. 영적/마법 톤.
  • E · Editorial Serif Display — 입체 그림자 + Fraunces italic + 골드. 뉴트로/인쇄물 톤.
  • F · Signature Bar Pill — 좌 시그니처 bar + bilingual + 우 모노 shortcut 영역. 정제된 디테일.

스펙트럼 — 미니멀 럭셔리 (A/E) ↔ 시스템 매칭 (B/F) ↔ maximalist 마법 (C/D)
