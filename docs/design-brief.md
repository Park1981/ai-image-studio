# AI Image Studio — Design Brief

## 프로젝트 소개

**AI Image Studio**는 로컬 PC에서 실행되는 AI 이미지 생성 웹 앱입니다.
ComfyUI(이미지 생성 엔진)와 Ollama(로컬 LLM)를 직관적인 웹 UI로 래핑하여,
복잡한 노드 에디터 없이도 누구나 고품질 AI 이미지를 쉽게 만들 수 있게 합니다.

- **사용 환경**: Windows 11 데스크탑 (로컬 전용, 배포 없음)
- **주 사용자**: 본인 (개발자, AI 이미지 생성에 관심 있는 1인 사용자)
- **사용 맥락**: 크롬 브라우저에서 사용. 듀얼 모니터 또는 단일 모니터.

---

## 핵심 기능 요약

### 1. 프롬프트 입력 & AI 보강
- 한국어로 이미지를 설명하면 AI가 영어 프롬프트로 자동 번역
- 품질 태그, 스타일 키워드 자동 추가 (AI 보강)
- 네거티브 프롬프트 자동 생성
- 보강 결과를 사용자가 직접 편집 가능

### 2. 이미지 생성
- 텍스트 → 이미지 생성 (1~4장 동시 배치)
- 결과를 2x2 그리드로 비교
- 실시간 진행률 표시
- 생성 중 취소 가능

### 3. 모델 & LoRA 설정
- 체크포인트 모델 선택 (드롭다운)
- LoRA 다중 적용 + 각각 강도 슬라이더
- 샘플러, 스케줄러, VAE 선택
- Steps, CFG, Seed, 이미지 크기 조절

### 4. 히스토리
- 생성 이력 저장, 검색, 필터링
- 과거 설정으로 즉시 재생성

### 5. 프로세스 상태 표시
- 헤더에 Ollama / ComfyUI 실행 상태 표시 (🟢 실행 중 / 🔴 중지)
- VRAM 사용량 표시 (선택)

## 화면 구성

### 페이지 3개
| 페이지 | 경로 | 용도 |
|--------|------|------|
| 메인 (생성) | `/` | 프롬프트 입력 + 설정 + 이미지 생성/결과 |
| 히스토리 | `/history` | 과거 생성 이력 목록, 검색, 재생성 |
| 설정 | `/settings` | 앱 설정 (URL, 프로세스 관리, 기본값, 테마) |

## 디자인 방향

### 분위기 & 레퍼런스
- **다크 모드 기본** (OLED 최적화, 눈 피로 감소)
- Midjourney 웹, ComfyUI, Automatic1111 같은 AI 이미지 생성 도구의 분위기
- 깔끔하고 미니멀, 이미지가 주인공이 되는 UI
- 불필요한 장식 요소 없음. 기능적이고 도구적인 느낌
- 애니메이션은 최소한 (기능적 전환만)

### 색상 방향
- 배경: 깊은 다크 (#0a0a0a ~ #141414 계열)
- 카드/패널: 배경보다 약간 밝은 다크 (#1e1e1e ~ #262626)
- 테두리: 미세한 구분선 (#27272a)
- 텍스트: 밝은 화이트 (#fafafa) + 보조 그레이 (#a1a1aa)
- 액센트: 인디고/퍼플 계열 (생성 버튼, 활성 상태 등)
- 상태 색상: 초록(실행), 빨강(중지/에러), 노랑(경고), 파랑(정보)

### 타이포그래피
- 한국어 지원 필수 (Pretendard 또는 시스템 폰트)
- 모노스페이스: 프롬프트 입력, 시드 번호 등에 사용
- 전체적으로 작은 텍스트 위주 (정보 밀도가 높은 도구형 UI)

## 필요한 UI 컴포넌트

### 기본 컴포넌트
- **Button**: Primary(인디고), Secondary(다크), Ghost, Icon 버튼
- **Input**: 텍스트 입력, Textarea(프롬프트용)
- **Slider**: 값 조절용 (LoRA 강도, Steps, CFG 등)
- **Dropdown/Select**: 모델, 샘플러 선택
- **Toggle/Switch**: 설정 온오프
- **Card**: 이미지 카드, 설정 패널 카드
- **Badge**: 상태 인디케이터 (🟢/🔴)
- **Toast**: 에러/성공 알림 팝업
- **Progress Bar**: 생성 진행률

### 복합 컴포넌트
- **Image Grid**: 2x2 이미지 결과 그리드 (선택 가능, 호버 시 액션)
- **Prompt Input**: 프롬프트 입력 + AI 보강 버튼 + 생성 버튼 조합
- **LoRA Panel**: LoRA 선택 + 다중 추가 + 각각 강도 슬라이더
- **Settings Panel**: 우측 사이드바 전체
- **History Card**: 생성 이력 한 건 (썸네일 + 프롬프트 + 설정 요약)
- **Status Indicator**: 헤더의 프로세스 상태 표시
- **Fullscreen Viewer**: 이미지 클릭 시 오버레이 확대

- **아이콘**: Lucide Icons 또는 Heroicons

## 핵심 UX 포인트

1. **이미지가 주인공**: 결과 이미지 영역이 가장 크고 눈에 띄어야 함
2. **설정은 보조**: 우측 사이드바에 정리, 접기 가능
3. **원클릭 생성**: 프롬프트 입력 → 생성 버튼 클릭 → 끝. 최소 동선
4. **한국어 친화**: 모든 UI 텍스트는 한국어, 프롬프트만 AI가 영어로 변환
5. **상태 투명성**: 생성 중/대기 중/에러 등 현재 상태가 항상 명확하게 보임
6. **키보드 친화**: Ctrl+Enter(생성), Escape(취소) 등 단축키 지원
